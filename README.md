
# WeRateDogs data analysis

## 概要

清洗 WeRateDogs 推特数据，创建有趣且可靠的分析和可视化。这份推特只包含基本的推特信息，要达到 "Wow!" 的效果，在分析和可视化前，还需要收集额外的数据、然后进行评估和清洗。


## 步骤

+ 数据整理，其中包括：
  - 收集数据
  - 评估数据
  - 清洗数据
  - 对清洗过的数据进行储存、分析和可视化
  
+ 书面报告 

1) 数据整理工作  

2) 数据分析和可视化

## 需求

+ Python 库
   - pandas
   - numpy
   - requests
   - tweepy
   - json

+ 创建包含图片的书面文档，并且把这些文档导出为 PDF 文件 如Jupyter Notebook
+ 文本编辑器，如免费软件 Sublime
